Windows 10 Development for Absolute Beginners

Bob Tabor
0% 完成
+

Loading

If you'd like to create Windows 10 apps on the Universal Windows Platform (UWP) and you have some experience with C#, these sessions are for you. Follow along with the coding exercises, as you learn key techniques and concepts.

 

Join the popular Bob Tabor for an exploration of XAML, layout controls, grids, Cortana integration, monetizing your apps with ads, and much more. Plus, test your new knowledge and skills with a series of challenges and assessments. Don’t miss this Windows 10 dev training to skill up on UWP app development.
1 | Series Introduction
Bob explains the purpose and intended audience for this series of lessons, what to expect, how to get the most out of it, what tools you need, and prerequisites to get started.
2 | Creating Your First Universal Windows Platform App
See how to create user interfaces by utilizing the Toolbox and the special XAML designer pane. Handle an event, and write C# to react to that event.
3 | Overview of Topics Related to Universal Windows Platform App Development
With the Hello World app completed, Bob uses experience to introduce you to many high-level concepts which are helpful as you build larger and more interesting apps.
4 | What is XAML?
XAML is one of the primary languages used to create user interfaces for Window 10 apps. Bob demonstrates the absolute basics of writing XAML, how it's related to XML, and why that's significant.
5 | Understanding Type Converters
Bob explains how an intelligent XAML parser allows you to create user interfaces much more easily and succinctly than when in C#, by converting simple strings into object instances or enums.
6 | Understanding Default Properties, Complex Properties, and the Property Element Syntax
Learn how an intelligent XAML parser allows you to keep your XAML succinct and tidy by inferring many elements and properties from their usage.
7 | Understanding XAML Schemas and Namespace Declarations
Bob tackles "the top of the page," where the XAML namespace declarations are generated, by default, in a new page template. Find out how they help the producer and consumer of the XML to communicate.
8 | XAML Layout with Grids
Learn how to create row definitions and column definitions and to then specify how each control can ask to be placed inside of a given cell in a grid.
9 | XAML Layout with StackPanel
Explore the StackPanel, which allows you to stack controls vertically or horizontally, and watch Bob demonstrate just how powerful this approach can be, using some advanced examples.
10 | Cheat Sheet Review: XAML and Layout Controls
Do you remember viewing a video that explained or demonstrated a technique in code, but you don't want to re-watch the entire video series to find it again? This cheat sheet review video is for you.
11 | Laudable Layout Challenge
Ready to flex your newfound knowledge about grids? Try completing this first challenge. These challenges help to reinforce the new information and skills you're learning in this course.
12 | Laudable Layout Challenge: Solution
Watch this video to see one possible solution to the Laudable Layout Challenge.
13 | Legendary Layout Challenge
Next up, can you shape and mold the XAML to produce the layout of this challenge with the StackPanel?
14 | Legendary Layout Challenge: Solution
Watch this video to see one possible solution to the Legendary Layout Challenge.
15 | Laborious Layout Challenge
So, you've got two challenges under your belt. This third challenge requires you use the Grid and StackPanel together, like you will often do in real apps.
16 | Laborious Layout Challenge: Solution
This is the solution video (one possible solution) to the Laborious Layout Challenge.
17 | XAML Layout with RelativePanel
Learn about new layout controls in Windows 10. Explore the RelativePanel, which allows you to align edges of controls to edges of other controls or the window itself.
18 | XAML Layout with the SplitPanel
Take a look at the SplitPanel control, which is layout control that helps you implement the hamburger-style navigation. Bob demonstrates how to show and hide and to allow the user to peek at an icon.
19 | Working with Navigation
Learn how to use navigation classes in the Universal Windows Platform, and explore methods to display different pages to users and to pass data from one page to another.
20 | Common XAML Controls: Part 1
There are so many controls in the Toolbox that we haven't used yet. Watch this video to explore tools for user input options.
21 | Implementing a Simple Hamburger Navigation Menu
Now that we understand how to utilize the RelativePanel and SplitPanel, and we have access to new controls in the Toolbox, we can build a "poor man's" hamburger-style navigation system.
22 | Cheat Sheet Review: Windows 10 Layout Hamburger Navigation and Controls
Do you remember viewing a video that explained or demonstrated a technique in code, but you don't want to re-watch the entire video series to find it again? This cheat sheet review video is for you.
23 | Hamburger Heaven Challenge
You've seen Bob create the elements of a hamburger navigation menu, search panel, back button, and more. Now it's your turn! Can you build a heavenly hamburger app?
24 | Hamburger Heaven Challenge: Solution
Watch this video to see one possible solution to the Hamburger Heaven Challenge.
25 | Common XAML Controls: Part 2
In this second installment of a tour through the Toolbox, learn about date input controls, progress bars and rings, and more.
26 | Working with the ScrollViewer
Get the details about how to use the ScrollViewer, and see a little "gotcha" that can happen when you work with the StackPanel and the ScrollViewer together.
27 | Canvas and Shapes
See how to order the shapes (and other XAML controls) on the Z index in a 3D plane.
28 | XAML Styles
Explore styles, which allow us to gather up common settings to apply across many XAML controls on a single page or across the entire app. Give them a name, and then apply them in a more succinct way.
29 | XAML Themes
As developers, we can utilize colors in our application to make it seem like it belongs among the other apps. Explore colors, high-contrast themes, and more.
30 | Cheat Sheet Review: Controls, ScrollViewer, Canvas, Shapes, Styles, Themes
Do you remember viewing a video that explained or demonstrated a technique in code, but you don't want to re-watch the entire video series to find it again? This cheat sheet review video is for you.
31 | Stupendous Styles Challenge
Take this challenge to create a different style of Windows 10 app! The primary requirement is to use and re-use as many styles as possible. How many styles can you come up with?
32 | Stupendous Styles Challenge Solution: Part 1: MainPage
Watch this video to see one possible solution to the Stupendous Styles Challenge.
33 | Stupendous Styles Challenge Solution: Part 2: Navigation and DonutPage
Watch this second solution video to see another possible solution to the Stupendous Styles Challenge.
34 | Stupendous Styles Challenge Solution: Part 3: CoffeePage
Watch this third solution video for another possible solution to the Stupendous Styles Challenge.
35 | Stupendous Styles Challenge Solution: Part 4: SchedulePage
Watch this fourth solution video for another possible solution to the Stupendous Styles Challenge.
36 | Stupendous Styles Challenge Solution: Part 5: CompletePage
Watch this fifth and final solution video to see another possible solution to the Stupendous Styles Challenge.
37 | Utilizing the VisualStateManager to Create Adaptive Triggers
See a demonstration of the basic building block required to create layouts which are adaptive to the screen resolution that they're running on.
38 | Working with Adaptive Layout
Utilizing adaptive triggers (learned in the previous lesson), find out how to think in terms of adaptive layouts by moving grids or other layout controls to entirely different cells.
39 | Adaptive Layout with Device Specific Views
Learn about adaptive layout, which helps when the desired layout is so different between devices that it would be cumbersome to utilize the VisualStateManager.
40 | Data Binding to the GridView and ListView Controls
Learn about the concept of data binding and how to bind a special Grid and List control to display the data in a tabular or list fashion. Plus, get the details on creating data templates.
41 | Keeping Data Controls Updated with ObservableCollection
Learn about a different collection which notifies all controls that are bound to it that it has new data to be displayed.
42 | Utilizing User Controls as Data Templates
Explore User Controls, which are most helpful for changing the size of the data template for a GridView or ListView when employing an adaptive layout.
43 | Cheat Sheet Review: Adaptive Layout, Data Binding
Do you remember viewing a video that explained or demonstrated a technique in code, but you don't want to re-watch the entire video series to find it again? This cheat sheet review video is for you.
44 | Adeptly Adaptive Challenge
In this final challenge, create a more full-featured news app, complete with hamburger style navigation, data binding, adaptive layout, and more. Good luck! You can do it!
45 | Adeptly Adaptive Challenge Solution: Part 1: Setup and MainPage Layout
Watch this video to see one possible solution to the Adeptly Adaptive Challenge.
46 | Adeptly Adaptive Challenge Solution: Part 2: Creating the Data Model and Data Binding
Watch this second video for another possible solution to the Adeptly Adaptive Challenge.
47 | Adeptly Adaptive Challenge Solution: Part 3: Creating a User Control as the Data Template
Watch this third video to see another possible solution to the Adeptly Adaptive Challenge.
48 | Adeptly Adaptive Challenge Solution: Part 4: Adaptively Resizing
Watch this fourth and final video to see another possible solution to the Adeptly Adaptive Challenge.
49 | UWP SoundBoard: Introduction
At this point, we're roughly halfway through the course. In the second half, Bob changes the format. Watch as Bob creates four entire apps from scratch employing the covered techniques.
50 | UWP SoundBoard: Setup and MainPage Layout
In this first video, learn from Bob as he sets up the project, copies the assets into the project, and creates the overall layout of the app's user interface.
51 | UWP SoundBoard: Creating the Data Model and Data Binding
In this second video, watch Bob create a class representing each sound, populate a collection of the Sound class, and bind it to a GridView.
52 | UWP SoundBoard: Playing Sounds with the Media Element
In this third video, see Bob handle the ItemClick event in the GridView to play the media related to the clicked item.
53 | UWP SoundBoard: Adding Drag and Drop
In this fourth video, Bob demonstrates how to add Drag and Drop to the application to allow a user to drag any sound file from their desktop into the app to play it back.
54 | UWP SoundBoard: Finishing Touches
In this fifth video, Bob cleans up the user interface and the code, putting the finishing touches on it so that it can be submitted to the Windows Store.
55 | UWP SoundBoard: Adding Assets with the Package.AppXManifest
In this sixth video, Bob explains the purpose of the Package.AppXManifest and demonstrates how to reference the artwork for the app to display a splash screen. Plus, he references the store artwork.
56 | UWP SoundBoard: Submitting to the Windows Store
Bob shows you where to go, how to upload the final package, and generally how easy it is to get your app into the Windows Store.
57 | UWP Weather: Introduction
Bob kicks off the start of a new project, a cute weather app pulling current weather from an online weather service, utilizing the user's device to determine his or her location.
58 | UWP Weather: Setup and Working with the Weather API
In this second video, Bob gets started by creating the project and focusing on how to retrieve data from the weather API. He demonstrates how to deserialize the JSON returned from the weather service.
59 | UWP Weather: Accessing the GPS Location
Now that we can call the web service and get data, this third video demonstrates how to access the current location of the user's device using the Location capabilities of Windows 10.
60 | UWP Weather: Testing Location in the Phone Emulator
In this third video, Bob uncovers a bug in the application by utilizing the Maps feature of the Phone Emulator. You can instruct the Emulator to act as if the user is in a different physical location.
61 | UWP Weather: Updating the Tile with Periodic Notifications
In this fourth video, Bob shows how to update the content of a Live Tile programmatically and then how to schedule future updates with live data, by creating a web service built with ASP.NET MVC.
62 | UWP Weather: Finishing Touches
In this fifth and final video, Bob cleans up the user interface and more.
63 | Album Cover Match Game: Introduction
Bob kicks off a new project, a familiarity game that uses the user's Music Library to test the user's familiarity with the songs they own. Learn how to access special libraries in Windows 10.
64 | Album Cover Match Game: Setup and Working with Files and System Folders
In this first video, Bob sets up the project and focuses on getting a reference to all the song files in the user's Music Library.
65 | Album Cover Match Game: Layout, Data Binding, and Game Setup
In this second video, Bob turns his attention to the layout for displaying the album covers for the randomly selected songs.
66 | Album Cover Match Game: Employing Game Logic
In this third video, Bob refocuses on the game logic, filling in some core missing pieces, like music playback and more.
67 | Album Cover Match Game: Evaluating User Input and Tracking Progress
In this fourth video, Bob handles the ItemClick event and evaluates the chosen album against the song that was playing.
68 | Album Cover Match Game: Enabling the Play Again Feature
In this fifth video, Bob employs the logic necessary to play a new round, which requires some refactoring and the setting of variables back to an initially valid state.
69 | Album Cover Match Game: Monetizing with Ads
Bob shows how to install, configure, and utilize the control used to display the ad placeholder on the user interface. He also discusses the settings required when submitting the app to Windows Store.
70 | Album Cover Match Game: In-App Purchase for Ad Removal
In this seventh and final video, Bob shows how to include in-app purchases to enable or disable some feature of your app or to sell consumable items in the app.
71 | Hero Explorer: Introduction
Bob kicks off a new project utilizing Marvel's API for exploring their universe of characters and comic books. In this lesson, learn how to sign a web API request with an MD5 hash.
72 | Hero Explorer: Accessing the Marvel Web API
In this video, Bob sets up the project but focuses on how to make calls and how to filter results. He uses the resulting JSON from a sample call to create a graph of classes representing the data.
73 | Hero Explorer: Creating an MD5 Hash and Calling the API
Bob demonstrates how to make a call to the Marvel API by creating a hash of the public and private keys and a timestamp, along with special cryptography classes in the framework.
74 | Hero Explorer: DataBinding and Navigating Through the Object Graph
In this fourth video, Bob provides the basic layout controls and databinds the "master" ListView to the list of Characters returned from the web API call.
75 | Hero Explorer: Displaying Character Details
In this fifth video, Bob handles the "master" ListView's ItemClick event to display the details about a given Character.
76 | Hero Explorer: Displaying Comic Books for a Character
In this sixth video, Bob displays a list of recent comics for the selected Character by making a new call to the Marvel API, creating new classes, and more.
77 | Hero Explorer: Displaying Comic Book Details
In this seventh video, Bob responds to the comic book's GridView ItemClick event to display the details about a given comic book.
78 | Hero Explorer: Adding an Adaptive Layout and UI Cleanup
In this eighth video, Bob adds the VisualStateManager and adaptive triggers to adaptively reshape the layout, based on the window size. He also performs other cleanup tasks on the user interface.
79 | Hero Explorer: Cortana Integration
In this ninth and final video, Bob adds Cortana integration to allow the user to refresh the list of characters or to display details about one of the heroes in the list.
80 | Series Wrap-Up
In this final video of the series, Bob congratulates you and shows you were to go for more information about the Universal Windows Platform.
